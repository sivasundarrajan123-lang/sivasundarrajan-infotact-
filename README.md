# PyChronicle

## Project Overview
PyChronicle is a Python-based Time-Travel Debugger that analyzes Python source code using the Abstract Syntax Tree (AST). The project aims to help developers understand code execution and debugging through static analysis and runtime tracing.

## Features
- Parse Python source code using AST
- Extract variables
- Analyze program structure
- Future support for execution tracing
